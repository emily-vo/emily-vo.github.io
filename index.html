Mini Minecraft

This is a simplified version of Minecraft I built in C++ and the OpenGL API, alongside two teammates, Ricky Rajani and William Ho. The project was completed over the span of three weeks as a final project in Spring 2016, and was my first time building a game with advanced graphics techniques.

Role:

  • Co-Developer

Features Implemented:

  • Octree Acceleration Structure and Visualization
  • Physics Implementation
  • Procedural Weather Generation
  • Raytracing Intersections for Block Interaction
  • Player Controls

Wikipedia Search Engine

This was a final project to exercise fundamental Graph Theory concepts and algorithms taught in my first Algorithms and Data Strucures class. The task at hand was to create a graph of Simple Wikipedia's pages and their outbound links, on which we performed various graph algorithms to observe various real-world applications, such as page rank, centrality, and connected components, and search ranking.

Role:

  • Solo Developer

Features Implemented:

  • XML Dump File Parser with Graph Generation
  • Dijkstra's, Non-Recursive Depth First Search, and Connected Components
  • Centrality, Page Rank, Search Ranking
