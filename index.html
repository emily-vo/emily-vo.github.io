Wikipedia Search Engine

This was a final project built in Java to exercise fundamental Graph Theory concepts and algorithms taught in my first Algorithms and Data Strucures class in Spring 2016. The task at hand was to create a graph of Simple Wikipedia's pages and their outbound links, on which we performed various graph algorithms to observe various real-world applications, such as page rank, centrality, and connected components, and search ranking.

Role:

  • Solo Developer

Features Implemented:

  • XML Dump File Parser with Graph Generation
  • Dijkstra's, Non-Recursive Depth First Search, and Connected Components
  • Centrality, Page Rank, Search Ranking
